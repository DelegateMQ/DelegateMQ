# DelegateMQ C# Interop Library

This is the thin .NET wrapper library for DelegateMQ. It communicates with C++ applications using the shared native DLL.

Full documentation for architecture and usage:
👉 **[docs/INTEROP.md](../../docs/INTEROP.md)**

## Setup
1.  **Build Native DLL**: You must build the DLL in `interop/native` first.
2.  **NuGet**: Add the `MessagePack` package to your project.
